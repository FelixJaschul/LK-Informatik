## Project Setup

### Compiled Files (.class)
This project is configured to store all compiled .class files in the `/out` directory. This directory is:
- Excluded from Git tracking (via .gitignore)
- Set as the output directory in IntelliJ IDEA

### Cleaning the Project
To remove all compiled files:
1. In IntelliJ IDEA, go to **Build** → **Build Project** (or press Ctrl+F9)
2. Then go to **Build** → **Rebuild Project**

Alternatively, you can manually delete the `/out` directory when needed.
